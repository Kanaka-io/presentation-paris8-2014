    Les limites du paradigme impératif

    Le principe de la programmation impérative, c'est la mutation d'état partagé

    La mutabilité rend la programmation concurrente extrèmement difficile

    Dans un contexte distribué et parallèle, la performance pure perd de l'importance

    Plus subtilement, les compilateurs de langages impératifs ont des typesystems plus "faibles" que les langages fonctionnels à typage statique

    Exemple

    Java

    
public class BankAccount {
    private BigDecimal balance;

    // ctr, getters, setters

    public void withdraw(BigDecimal amount) throws IllegalStateException {
        if (amount.compareTo(balance) <= 0){
            balance = balance.substract(amount);
        } else {
            throw new IllegalStateException("Insufficient funds");
        }
    }

    public void performWithdrawals(List<BigDecimal> withdrawals)
                            throws IllegalStateException {
        BigDecimal totalDebit = BigDecimal(0)
        for(BigDecimal op : withdrawals) {
            totalDebit = totalDebit.add(op)
        }
        withdraw(totalDebit)
    }
}

    Scala

    
case class BankAccount(balance: BigDecimal){
  def withdraw(amount: BigDecimal): String \/ BankAccount =
    if (amount <= balance) Right(copy(balance - amount))
    else Left("Insufficient funds")

  def performWithdrawals(withdrawals: List[BigDecimal]) =
    withdraw(withdrawals.sum)
}

    Global State

    Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the background.

    Custom Events

    Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 

    
Reveal.addEventListener( 'customevent', function() {
    console.log( '"customevent" has fired' );
} );

    Slide Backgrounds

    Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.

    Background Transitions

    Pass reveal.js the backgroundTransition: 'slide' config argument to make backgrounds slide rather than fade.

    Background Transition Override

    You can override background transitions per slide by using data-background-transition="slide".
